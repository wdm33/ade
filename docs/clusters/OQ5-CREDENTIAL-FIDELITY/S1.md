## Mechanical Acceptance Criteria
- **AC-1:** `StakeCredential` is the 2-variant enum + `hash()`; both decoders map

  fingerprint test `discriminant_changes_fingerprint`.

  --workspace` green. No NEW test failures vs baseline (pre-existing env-blocked:
  boundary_fingerprint/anchor_hashes absent snapshots; ade_plutus aiken fixture).

## Hard Prohibitions
- No `Hash28 ŌåÆ StakeCredential` coercion (only `decode` and explicit
  `KeyHash`/`ScriptHash` constructors create credentials). `.hash()` is read-only
  extraction, never used to re-key authoritative cert/gov state.
- No `(tag, Hash28)` tuple key. No tag drop at decode. No Byron / withdrawal /
  required-signer credential changes (non-goals).
