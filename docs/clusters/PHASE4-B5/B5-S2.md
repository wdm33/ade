## Slice-Entry Proof Obligations (gating)

- **PO-1 (no CertState access).** `apply_conway_gov_cert` must not take `CertState`
  — owner exclusivity (CE-2) is enforced structurally, not by convention. The B4
  delegation/pool half of composites 10/12/13 is applied by `apply_conway_cert`
  in the same pass (B5-S3) and must not be re-applied here.
- **PO-2 (credential projection).** `StakeCredential(pub Hash28)` and the
  `ConwayGovState` maps key by `Hash28`. The projection is `cred.0` (`Hash28` is
  `Clone`, not `Copy` — use `.clone()`). No key/script discriminant exists at
  this layer (pre-existing; OQ-5).
- **PO-3 (env laziness).** Only tags 16/18 read the env. `env: Option<&GovCertEnv>`;
  env-free arms ignore it. This honors CE-3's `env_free_gov_certs_need_no_env`
  and makes the absent-param fail-fast occur exactly when a DRep expiry is needed.
