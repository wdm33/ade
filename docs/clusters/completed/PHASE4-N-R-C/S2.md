# Slice C2 — Shelley genesis closed-contract parser

**Cluster:** PHASE4-N-R-C.
**Strengthens:** `CN-PROD-02` (genesis parser sub-item).
**Introduces:** `ade_runtime::producer::genesis_parser` +
flips `CN-GENESIS-01` and `DC-GENESIS-01` to `enforced`.

## §1 Intent

Land the real Shelley genesis closed-contract parser.
Consumes the OQ7 fixtures captured in N-R-A A1.

Important: per A1's OQ7 audit, the load-bearing slot/KES
geometry fields live in **Shelley** genesis, not Conway.
Conway genesis carries governance parameters not consumed
by `GenesisAnchor`.

## §2 Closed required-field set

| JSON key | Type | Maps to `GenesisAnchor` |
|---|---|---|
| `networkMagic` | u32 (strict number) | `network_magic` |
| `systemStart` | ISO 8601 UTC | `slot_zero_time_unix_ms` |
| `slotLength` | u64 (seconds) | `slot_length_ms` (× 1000) |
| `slotsPerKESPeriod` | u64 | `slots_per_kes_period` |
| `maxKESEvolutions` | u32 | `kes_max_period` |
| (caller-supplied) | u64 | `kes_anchor_slot` |

## §3 Hard rules (DQ-C2)

- Required fields fail-closed on missing / malformed /
  wrong-type input.
- No implicit defaults.
- No stringly fallback (`"1"` rejected for u32).
- Extra unknown keys accepted-and-ignored, iff inert
  (byte-identical `GenesisAnchor` for canonical vs.
  extras-added fixtures).

## §4 ISO 8601 → Unix epoch ms conversion

`parse_iso8601_to_unix_ms("YYYY-MM-DDTHH:MM:SSZ")` uses
Howard Hinnant's `days_from_civil` algorithm —
deterministic, no chrono/time crate dependency. Anchored to
known values (`1970-01-01T00:00:00Z` → 0;
`2022-06-01T00:00:00Z` → 1654041600000).
