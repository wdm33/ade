## 4. Intent

logic. A block's reject reason is a closed enum; the *canonical/replay
comparison surface* is a coarse closed **reject class** (oracle-comparable),
while the full nested error is carried for debugging only.

## 9. Design Summary

```rust
// ade_ledger::block_validity::verdict
use ade_core::consensus::events::Point;
use ade_core::consensus::errors::HeaderValidationError;
use ade_types::{BlockNo, Hash28, Hash32};
use crate::error::LedgerError;
use crate::rules::BlockVerdict;

/// The verdict. Valid carries the body stats; Invalid carries a coarse,
/// oracle-comparable `class` PLUS the full structured error for debugging.
#[derive(Debug, Clone, PartialEq, Eq)]
pub enum BlockValidityVerdict {
    Valid   { tip: Point, block_no: BlockNo, body: BlockVerdict },
    Invalid { class: BlockRejectClass, error: BlockValidityError },
}

/// Coarse, closed reject class — the canonical/replay comparison surface and
/// what the reference oracle exposes. CBOR-round-trippable.
#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum BlockRejectClass {
    HeaderInvalid,
    BodyInvalid,
    BodyHashMismatch,
    MalformedField,
    MissingConsensusInput,
}

/// Full structured reject reason. Closed; no owned String, no Box<dyn>.
#[derive(Debug, Clone, PartialEq, Eq)]
pub enum BlockValidityError {
    Header(HeaderValidationError),         // from ade_core (N-B)
    Body(LedgerError),                     // from ade_ledger (Phases 1-3)
    BodyHashMismatch { header: Hash32, actual: Hash32 },
    MalformedField(FieldError),
    MissingConsensusInput(MissingInput),
}
impl BlockValidityError {
    pub fn class(&self) -> BlockRejectClass;   // total mapping
}

#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub struct FieldError { pub field: FieldKind, pub expected: usize, pub actual: usize }

/// Closed set of fixed-size fields whose length is checked fail-closed.
#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum FieldKind {
    VkeyWitness, BootstrapKey, Ed25519Signature, VrfVkey, VrfProof,
    KesVkey, KesSignature, OpCertSignature, BlockBodyHash,
}

#[derive(Debug, Clone, PartialEq, Eq)]
pub enum MissingInput {
    EpochNonce, SetSnapshot, PoolVrfKeyhash(Hash28), ActiveSlotsCoeff,
}
```

> `FieldKind` is a **closed enum**, not a `&'static str` — so the comparison
> surface is round-trippable and truly closed (mirrors N-B's `DecodeError`
> discipline). The full `LedgerError`/`HeaderValidationError` detail is NOT
> CBOR-encoded (those are large/unstable); only the **comparison surface** is.

### Canonical encoding (comparison surface only)

```rust
// ade_ledger::block_validity::encoding
/// Encode the replay/comparison surface: Valid -> [0, tip, block_no];
/// Invalid -> [1, reject_class_discriminant]. The full error detail is
/// debug-only and NOT part of the canonical bytes.
pub fn encode_verdict_surface(v: &BlockValidityVerdict) -> Vec<u8>;
pub fn decode_verdict_surface(bytes: &[u8]) -> Result<VerdictSurface, SurfaceDecodeError>;

#[derive(Debug, Clone, PartialEq, Eq)]
pub enum VerdictSurface { Valid { tip: Point, block_no: BlockNo }, Invalid { class: BlockRejectClass } }

#[derive(Debug, Clone, PartialEq, Eq)]
pub enum SurfaceDecodeError { Cbor(&'static str), UnknownDiscriminant { for_enum: &'static str, found: u32 }, FieldCount { expected: u32, actual: u32 } }
```

`encode_verdict_surface` is what B1-S6/S7 replay and compare; it is byte-stable
and oracle-aligned (the oracle gives Valid/Invalid + class, not Ade internals).

## 14. Hard Prohibitions

### Inherited (cluster.md)
- No owned `String` in error enums; no `#[non_exhaustive]`; no `Box<dyn>`;
  no `HashMap`/float/wall-clock.

### Slice-specific
- `FieldKind` is a closed enum, never a `&'static str` field carrying a name.
- Do NOT CBOR-encode the full `LedgerError`/`HeaderValidationError` — only the
  comparison surface.
- No transition logic.
