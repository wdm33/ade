# Slice B3 — seed_to_snapshot adapter + no-refscript-skip CI

**Cluster:** PHASE4-N-M-B
**CEs:** CE-N-M-B-2 (CN-ADMIT-02), CE-N-M-B-5 (DC-ADMIT-09)
**Dependencies:** none (uses N-M-A surfaces directly)
**Registry effects on merge:** CN-ADMIT-02 + DC-ADMIT-09 → `enforced`.

## Intent

Sole authority composing N-M-A's seed import outputs into a
persisted snapshot at `seed_point.slot` so
`bootstrap_initial_state`'s warm-start branch (CN-NODE-01)
picks it up. The bridge does NOT bypass `bootstrap_initial_state`.

Hard non-goal: this slice MUST NOT add any partial
reference-script support or seed-import fallback (DC-ADMIT-09).
A1's fail-fast on `JsonSeedError::UnsupportedTxOutFeature`
stays as-is.

## Scope

- `crates/ade_node/src/admission/seed_to_snapshot.rs` (new GREEN):
  - `pub fn seed_to_snapshot(utxo, chain_dep_seed, seed_point, store) -> Result<Hash32, SeedToSnapshotError>` — sole authority. Returns the initial ledger fingerprint computed at capture time.
  - Closed `SeedToSnapshotError` (`Encode`, `Store`, `LedgerBuild`).
  - Internally:
    1. Build an in-memory `LedgerState` from `(era=Conway, utxo, chain_dep_seed)`.
    2. Compute `fingerprint(&ledger).combined`.
    3. Call `PersistentSnapshotCache::capture(seed_point.slot, &ledger, &chain_dep)` (CN-STORE-08 sole encode authority).
    4. Return the fingerprint.
- `ci/ci_check_admission_no_refscript_skip.sh` — greps the
  admission code paths for any permissive match on
  `JsonSeedError::UnsupportedTxOutFeature` or any
  `referenceScript` skip / TODO.

## §12 Mechanical Acceptance Criteria

- [ ] `seed_to_snapshot_writes_via_persistent_cache` — captured
  snapshot bytes decode back via `PersistentSnapshotCache::nearest_le`.
- [ ] `seed_to_snapshot_returns_initial_ledger_fingerprint` —
  returned fp equals `fingerprint(built_ledger).combined`.
- [ ] `seed_to_snapshot_two_runs_byte_identical` — same inputs →
  same snapshot bytes + same returned fp.
- [ ] `seed_to_snapshot_propagates_pre_conway_encode_error_as_authority_fatal`
  (the underlying framing rejects pre-Conway).
- [ ] `ci/ci_check_admission_no_refscript_skip.sh` passes.

## §14 Hard Prohibitions

- No `JsonSeedError::UnsupportedTxOutFeature { feature: "referenceScript" } => _ /* skip */` pattern (DC-ADMIT-09).
- No `// TODO: refscript` / `// FIXME: refscript` / similar.
- No bypassing `PersistentSnapshotCache::capture` (CN-STORE-08).
- No second `pub fn seed_to_snapshot`.

## §15 Non-Goals

- No reading the seed JSON file (B4 reads + passes the
  `(UTxOState, UtxoFingerprint)` from N-M-A's importer).
- No bootstrap_initial_state call (B4 does the warm-start).
- No A1.1 work.
