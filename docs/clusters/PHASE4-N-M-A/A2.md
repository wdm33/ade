# Slice A2 — BootstrapAnchor type + canonical CBOR

**Cluster:** PHASE4-N-M-A
**CEs:** CE-N-M-A-3 (CN-ANCHOR-01 + DC-ANCHOR-01)
**Dependencies:** A1 (UtxoFingerprint)
**Registry effects on merge:** CN-ANCHOR-01 + DC-ANCHOR-01 → `enforced`.

## Intent

Mint a typed `BootstrapAnchor` carrying the full provenance of
an oracle-seed import. Closed record (no default impl, no
non_exhaustive); canonical CBOR round-trip; single mint
authority.

## Scope

- **BLUE** `crates/ade_ledger/src/bootstrap_anchor/anchor.rs`:
  - `pub struct BootstrapAnchor` (closed; all 6 fields pub).
  - `pub fn encode_bootstrap_anchor(&BootstrapAnchor) -> Vec<u8>`
    (canonical CBOR, SCHEMA_VERSION=1).
  - `pub fn decode_bootstrap_anchor(&[u8]) -> Result<BootstrapAnchor, BootstrapAnchorError>`.
- **BLUE** `crates/ade_ledger/src/bootstrap_anchor/error.rs`:
  - Closed `BootstrapAnchorError` (`Cbor`, `UnknownVersion`,
    `Structural`).
- **GREEN** `crates/ade_runtime/src/bootstrap_anchor.rs`:
  - `pub fn mint(import_inputs: MintInputs) -> BootstrapAnchor`.
  - `MintInputs` is a closed struct bundling all 6 fields.

## Struct shape

```rust
#[derive(Debug, Clone, PartialEq, Eq)]
pub struct BootstrapAnchor {
    pub network_magic: u32,
    pub genesis_hash: Hash32,
    pub seed_point: SeedPoint,
    pub seed_artifact_hash: Hash32,
    pub imported_utxo_fingerprint: UtxoFingerprint,
    pub initial_ledger_fingerprint: Hash32,
}

#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub struct SeedPoint {
    pub slot: SlotNo,
    pub block_hash: Hash32,
}
```

No `Default` impl on either type. No `#[non_exhaustive]`. The
struct's full-field-required initialization is the type-level
half of ¬P-A3.

## CBOR wire shape

```
array(7) [
  uint    SCHEMA_VERSION (= 1),
  uint    network_magic,
  bytes(32) genesis_hash,
  array(2) [ uint seed_point.slot, bytes(32) block_hash ],
  bytes(32) seed_artifact_hash,
  bytes(32) imported_utxo_fingerprint,
  bytes(32) initial_ledger_fingerprint,
]
```

## §12 Mechanical Acceptance Criteria

- [ ] `bootstrap_anchor_round_trips_via_canonical_cbor` —
  `decode(encode(a)) == a` for a hand-crafted anchor.
- [ ] `bootstrap_anchor_encode_two_runs_byte_identical` —
  determinism.
- [ ] `bootstrap_anchor_decode_rejects_unknown_version` —
  version=99 → `BootstrapAnchorError::UnknownVersion`.
- [ ] `bootstrap_anchor_decode_rejects_short_buffer` →
  `BootstrapAnchorError::Structural`.
- [ ] `mint_composes_inputs_byte_identically` — same
  `MintInputs` → same anchor.
- [ ] `ci/ci_check_bootstrap_anchor_closure.sh` — single pub
  `mint` in workspace; single pub
  `encode_bootstrap_anchor`/`decode_bootstrap_anchor` pair.

## §14 Hard Prohibitions

- No `Default` impl on `BootstrapAnchor` or `SeedPoint`.
- No `#[non_exhaustive]` attribute.
- No HashMap / float / wall-clock / rand.
- No second mint authority.

## §15 Non-Goals

- No on-disk persistence (the anchor lives inside the WAL
  header in A3).
- No mint helpers for "default network magic" etc. (operator
  must supply explicitly).
